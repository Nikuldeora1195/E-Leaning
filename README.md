# LearnMax

LearnMax is a full-stack MERN e-learning platform built by **Nikul Kumar**.

It supports:
- student registration, login, enrollment, learning progress, quizzes, and certificates
- teacher course creation, lesson management, announcements, and student tracking
- admin dashboard, user management, and course management

## Tech Stack

- Frontend: React, Vite, Tailwind CSS, React Router, Axios
- Backend: Node.js, Express.js, MongoDB, Mongoose
- Auth: JWT, bcrypt

## Local Run

### Backend

```bash
cd backend
npm install
# create backend/.env from backend/.env.example
npm start
```

### Frontend

```bash
cd eduvillage-frontend
npm install
npm run dev
```

## Main Flows

- Student: register, browse courses, enroll, open lessons, complete learning, attempt quiz, view certificate
- Teacher: create course, add sections and lessons, upload media, publish, post announcements
- Admin: view platform stats, manage users, manage courses
