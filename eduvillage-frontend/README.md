# LearnMax Frontend

Frontend app for **LearnMax**, an e-learning platform by **Nikul Kumar**.

## Stack

- React
- Vite
- Tailwind CSS
- React Router
- Axios

## Run Locally

```bash
cd eduvillage-frontend
npm install
npm run dev
```

The app expects the backend at:

```text
http://localhost:5000/api
```

## Current Areas

- Student dashboard and learning flow
- Teacher course and content management
- Admin dashboard, users, and courses
