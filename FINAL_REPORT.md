# LearnMax Project Summary

**Project Name:** LearnMax  
**Type:** Full-stack MERN e-learning platform  
**Built By:** Nikul Kumar

## Overview

LearnMax is a role-based learning platform where:
- students can enroll in courses, learn through lessons, track progress, take quizzes, and view certificates
- teachers can create and publish courses, manage sections and lessons, upload media, and post announcements
- admins can view platform stats, manage users, and manage courses

## Main Features

- JWT authentication
- student, teacher, and admin dashboards
- course creation and publishing
- section and lesson management
- image and video support
- quiz creation and quiz submission
- enrollment and progress tracking
- certificate view
- admin user management
- admin course management

## Tech Stack

- React + Vite
- Tailwind CSS
- Node.js + Express
- MongoDB + Mongoose
- JWT + bcrypt
